The ''1-my_list'' module
========================

using ''MyList''
----------------

First, import the ''MyList'' class from ''1-my_list'' 
>>> MyList = __import__("1-my_list").MyList

Then, use it:

    Test 00: check instantiation
    >>> print(MyList())
    []

    Test 01: append
    >>> list1 = MyList()
    >>> list1.append(1)
    >>> print(list1)
    [1]

    Test 02: print_sorted
    >>> list1 = MyList()
    >>> list1.append(2)
    >>> list1.append(1)
    >>> list1.print_sorted()
    [1, 2]

    Test 02: print_sorted, negative number
    >>> list1 = MyList()
    >>> list1.append(1)
    >>> list1.append(-1)
    >>> list1.print_sorted()
    [-1, 1]
    
    Test 02: print_sorted with empty list
    >>> list1 = MyList()
